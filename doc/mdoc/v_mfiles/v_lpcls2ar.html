v_lpcls2ar

PURPOSE

V_LPCLS2AR convert line spectrum pair frequencies to ar polynomial AR=(LS)

SYNOPSIS

function ar=v_lpcls2ar(ls)

DESCRIPTION 

V_LPCLS2AR convert line spectrum pair frequencies to ar polynomial AR=(LS)
 input vector elements should be in the range 0 to 0.5
